| Description                              | Angular development mode                            | Angular production mode                            |
| ---------------------------------------- | --------------------------------------------------- | -------------------------------------------------- |

| Run end-to-end tests headless once       | `nx run energy-insights-app-e2e:e2e:ci-development` | `nx run energy-insights-app-e2e:e2e:ci-production` |
